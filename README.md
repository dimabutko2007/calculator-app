# calculator-app
Simple calculator web app (HTML, CSS, JavaScript)

ОПИС
Створіть базовий Калькулятор.
Реалізуйте код відповідно до стандарту JavaScript – ES2015 (або новішої версії).

Основними методами Калькулятора є:

-арифметичні операції: +, -, *, /

-результат: =

-відсоток: % (результат — це відношення до поточного числа, виражене у частках від 100)

-скидання: C (скидання результату до значення за замовчуванням)

-знак: +/- (перемикання між додатним та від’ємним числом)

-десяткова крапка: . (використовується для відокремлення цілої частини від дробової у числі, записаному в десятковій формі)

Калькулятор має рахувати так само добре, як і будь-який сучасний калькулятор.

Ось деякі характеристики правильного Калькулятора (ви можете знайти й інші):

-значення в полі виводу за замовчуванням — 0

-поле історії за замовчуванням — порожнє

-знаки або цифри поточного значення додаються в поле виводу

Програма не повинна виконувати математичні операції з рядками (strings), а працювати виключно з числами.
